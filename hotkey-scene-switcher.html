Hotkey Scenes Switcher

This asset allows you to bind any scene in your project to particular hotkey from Shift+0 to Shift+9. So you can have up to 10 scenes to be accessed within a single click! No need to navigate folders anymore!


This is the first editor tool that I've made for personal use and now it's completely free and avalable on the Unity Asset Store!


Features:

  • Bind up to 10 scenes on hotkeys
  • Scenes available through the visual menu on the top
  • Save changes dialog - you'll not loose your changes!
  • Full source code available
  • And more features to come in the future...

How to use

To use this asset just import it and go to: Top Menu -> Vis -> Hotkey Scenes Switcher -> Edit. Or you can hit Shift+E hotkey. Edit window will pop up and from there you'll be able to adjust your hotkeys by binding specific scenes to hotkeys.
