Auto Image Cropper

This asset allows you to automatically crop any PNG texture with transparency while importing it into Unity!


Modes:

  • 1. Auto-crop while importing.
  • 2. Manual crop of any given texture.
  • 3. Manual crop of any textures in a given directory.

By default automatic cropping while importing is set to true - so any PNG-texture with transparency that you'll import will be cropped and saved near original uncropped version. You can also choose to rewrite original versions of files with cropped ones by choosing "Rewrite original file" option in settings.


Features:

  • Customizable padding
  • Multiple output file formats
  • Alpha threshold adjustable
  • Lots of tool-tips
  • Full source code available
  • Fully customizable output files naming rules
  • And more features to come in the future...

How to use

To use this asset just import it and you ready to go - your imported textures will be cropped while imported! No need for additional actions. However if you want you can use full power of this tool by going to Settings window: Top Menu -> Vis -> Auto Image Cropper.
