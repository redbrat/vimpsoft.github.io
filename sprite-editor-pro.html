Sprite Editor Pro

This asset will fit right for your needs if the standard Unity's Sprite Editor isn't enough for you. It's dedicated for difficult cases, particularly where you have an incorrectly or tricky layed out sprite atlases which you want to slice for separate sprites. It also supports visual tool for creating patterns to parse arbitrary atlas-describing texts.


Features:

  • Visually scriptable sprite slice logic
  • Two sprite slicing modes
  • Parse arbitrary-formatted layout describing text
  • Convenient reusable chunks of layout
  • Lots of possible settings
  • Different sprites layout orientation supported
  • Full source code available
  • Rich sprites preview experience
  • Store your layout settings in the preset files to reuse on multiple similar sprites
  • And more features to come in the future...
